Numero Secreto

Este código HTML cria uma página para um jogo em que o usuário tenta adivinhar um número secreto.

O conteúdo inclui um título, uma indicação do intervalo numérico (de 0 a 100) e uma área para exibir mensagens interativas sobre os palpites do usuário.

O layout é estilizado por um arquivo CSS externo e a funcionalidade do jogo é implementada por três scripts JavaScript, que lidam com o sorteio do número, reconhecimento de voz e validação dos palpites.
